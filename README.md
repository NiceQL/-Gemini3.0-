# -Gemini3.0-
纯个人分享过Gemini3.0绑卡教程
虚拟卡认证过程卡了我挺久，迫不及待分享一下我刚成功的自用方法，无需实体卡，分分钟就成功。

首先大家至少过了学生认证以及之前的步骤应该没问题吧，我先只讲过虚拟卡的方法了~
我使用的是一个在别的视频里看到的SAVO的虚拟卡，可能也有很多人尝试过这张卡但是不能用、没通过的，但按照我这种方法能够成功（我一次尝试成功了，希望你们也能复刻）
这个虚拟卡是很好申请的，Google play下载SAVO就行。当然商店地区需要是美区。不是美区的去删掉所有其他地区的支付资料，只留一个或新建一个美区的支付资料就可以改为美区了。（一定要一个一个关闭所有非美国的支付资料，然后再添加一个美国任意地区的支付资料）

这个虚拟卡APP的注册认证和申虚拟卡速度都很快，我申请时分分钟就OK。注册认证我就填的自己的真实信息。注册时要填邀请码才能获得一张免费虚拟卡激活券，这里就厚着脸皮大家填我的吧：M8IR7
过了最低的一级认证，申请完最低级的新星visa卡就可以了，点击卡号，不用充值，点继续查看卡号，复制一下卡号信息，日期，安全码就行。
接下来就是核心操作了！

打开手机Google Play或者谷歌钱包（Google Pay）,找到付款与订阅中的支付方式
添加卡，就填这张卡的信息就行，然后他要你邮箱或者手机接收验证码验证这张卡！快！停！下！不！要！验！证！

（手快不听指挥的请再绑定结束后取消绑定，然后重新再绑，因为若是验证成功后，他提示这张卡unavailable，不能用来订阅）

打开电脑浏览器，回到电脑的学生验证页面，订阅时你会发现有这张卡了，但还要验证，此时在电脑验证只需要填入卡号后面的3位数字安全码就行，无需手机、邮箱验证，而且验证完安全码便可以直接点击订阅-订阅成功！
